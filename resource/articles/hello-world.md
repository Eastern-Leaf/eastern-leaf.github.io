## 欢迎来到我的博客 🎉

这是我的第一篇博客文章。这个博客是一个**纯静态网站**，没有后端和数据库，所有文章都以 Markdown 格式存放在 `resource/` 文件夹中。

---

### 博客特点

- 🚀 **纯静态** — 无需服务器，直接用浏览器打开或部署到任意静态托管服务
- 📝 **Markdown 写作** — 所有文章使用 Markdown 格式，写作体验极佳
- 🎨 **简洁美观** — 三栏布局，阅读体验舒适
- 🏷️ **标签支持** — 可以为文章添加标签分类

### 如何发布新文章？

1. 在 `resource/` 文件夹中新建一个 `.md` 文件，写好文章内容
2. 编辑 `resource/article-list.json`，添加一条文章记录：

```json
{
    "id": "my-new-post",
    "title": "我的新文章标题",
    "date": "2026-07-08",
    "file": "my-new-post.md",
    "tags": ["标签1", "标签2"]
}
```

3. 刷新页面，新文章就会出现在左侧列表中！
