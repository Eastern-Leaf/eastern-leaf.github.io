## Markdown 语法速查

这是一份常用的 Markdown 语法参考，方便写作时查阅。

---

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

---

### 文本样式

- **粗体**：`**粗体**` 或 `__粗体__`
- *斜体*：`*斜体*` 或 `_斜体_`
- ~~删除线~~：`~~删除线~~`
- `行内代码`：`` `行内代码` ``

---

### 列表

无序列表：

```markdown
- 项目一
- 项目二
  - 子项目
```

有序列表：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

---

### 引用

> 这是一段引用文字。
> 
> 引用可以有多段。

语法：`> 引用内容`

---

### 代码块

````markdown
```javascript
function greet(name) {
    console.log(`你好，${name}！`);
}
```
````

效果：

```javascript
function greet(name) {
    console.log(`你好，${name}！`);
}
```

---

### 链接与图片

- 链接：`[文字](https://example.com)`
- 图片：`![替代文字](image.png)`

---

### 表格

```markdown
| 姓名 | 年龄 | 职业 |
|------|------|------|
| 张三 | 28   | 工程师 |
| 李四 | 32   | 设计师 |
```
