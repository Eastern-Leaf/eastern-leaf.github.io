## 正则表达式实用技巧

正则表达式是文本处理的利器。

### 常用元字符

| 元字符 | 含义 |
|--------|------|
| `^` | 开头 |
| `$` | 结尾 |
| `\d` | 数字 |
| `\w` | 单词字符 |
| `+` | 一个以上 |
| `*` | 零个以上 |

### 实战示例

```javascript
// 匹配邮箱
/^[\w.-]+@[\w.-]+\.\w+$/

// 匹配手机号（中国大陆）
/^1[3-9]\d{9}$/

// 提取 URL 参数
const params = Object.fromEntries(
    new URLSearchParams(location.search)
);
```

掌握正则可以大幅提升文本处理效率。
